## üåü Featured Projects (Ordered by Complexity)

* [**SPLADE-PT-BR**](https://github.com/AxelPCG/SPLADE-PT-BR): Implementation of the SPLADE (Sparse Lexical and Expansion) model for the Portuguese language. This project focuses on **Neural Information Retrieval**, using transformers to create sparse embeddings that allow for high-efficiency semantic search and term expansion.

* [**TOTVS-ATLAS**](https://github.com/Esparta-Solucoes/TOTVS-ATLAS): Collaborative development of an intelligence and automation system for the TOTVS ecosystem. It leverages **LLM Agents and RAG** (Retrieval-Augmented Generation) to optimize corporate workflows and data querying.

* [**geo-access-control**](https://github.com/AxelPCG/geo-access-control): A sophisticated security backend that implements geolocation-based access control. It processes real-time device metadata to identify and block suspicious logins based on distance-threshold algorithms, featuring integrations with GeoIP and Slack. (PT-BR).

* [**PROJETO_LANGCHAIN**](https://github.com/AxelPCG/PROJETO_LANGCHAIN): A lab for experimenting with the **LangChain framework**, including implementations of autonomous agents, vector database integrations, and complex chain logic for LLM applications. (PT-BR).

* [**Data-Science**](https://github.com/AxelPCG/Data-Science): A comprehensive repository containing various data science and machine learning projects. It covers exploratory data analysis (EDA), data visualization, statistical testing, and predictive modeling using Python and Scikit-Learn. (PT-BR).

* [**FECHADURA_ELETRONICA_INTELIGENTE**](https://github.com/AxelPCG/FECHADURA_ELETRONICA_INTELIGENTE): Development of an intelligent electronic lock based on the **CDIO methodology**. This project bridges Electronic Engineering and software, focusing on secure embedded systems and hardware prototyping. (PT-BR).

* [**github-stats-modified**](https://github.com/AxelPCG/github-stats-modified): A customized version of GitHub profile stats, featuring modified metric logic and styling to provide more accurate and visually appealing data for developer profiles.

* [**PROJETO-RDBMS-Oracle**](https://github.com/AxelPCG/PROJETO-RDBMS-Oracle): Full database modeling for the "Melhores Compras" e-commerce system. Includes Entity-Relationship Diagrams (ERD) and SQL implementation using Oracle Data Modeler and SQL Developer. (PT-BR).

* [**TRABALHOS-C**](https://github.com/AxelPCG/TRABALHOS-C): A collection of academic projects and fundamental algorithms implemented in C, covering data structures and memory management. (PT-BR).
